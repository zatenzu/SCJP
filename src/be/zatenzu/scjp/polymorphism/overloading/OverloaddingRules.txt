- Can change access modifier

===============================================================
- varType > instanceType (compilation time, not runtime)
- widening > boxing
- widening > var args
- boxing > var args
- widening + boxing = error
- widening + var args = ok
- boxing + var args = ok
- boxing + IS-A = ok
